# krasis 开发计划（AI 友好版）

## 概述

本文档为 AI 大模型提供开发指南。每个任务都标注了**需求来源文档**，AI 可根据文档路径找到对应的需求说明进行开发。

```
┌─────────────────────────────┐

│  📝 全部笔记                 │

│  ├─ 笔记标题 B              │
│  └─ ...                    │
├─────────────────────────────┤
│  🤖 AI 对话                 │
│  ├─ 对话 1                 │
│  └─ ...                    │
├─────────────────────────────┤
│  ⚙️ 设置                    │
│  👤 用户名                   │
└─────────────────────────────┘
```

**响应式布局**：

| 屏幕宽度 | 布局变化 |
|----------|----------|
| < 768px | 侧边栏默认隐藏，点击按钮展开为抽屉 |
| 768-1024px | 侧边栏折叠为图标模式（60px） |
| > 1024px | 侧边栏完整显示（240px） |

## 关键决策

| 问题 | 决策 | 备注 |
|------|------|------|
| 实时协同 | 乐观锁 + OT | `docs/backend/modules.md` (2.5节) |
| AI 模型配置 | **后台配置制** | 所有 LLM/Embedding 模型从数据库读取，支持多提供商，30 秒热更新 |
| AI 嵌入模型 | OpenAI → Ollama → 本地 | 可通过后台随时切换 |
| RAG 检索策略 | **混合检索** | Qdrant 向量检索优先，结果不足时 Postgres FTS (zhparser) 回退 |
| 部署环境 | Docker Compose (开发) / K8s (生产) | `docs/architecture/deployment.md` |
| 管理后台 | 独立管理后台页面 | `docs/backend/admin.md` |
| 分享审核 | 人工审核机制 | 分享需审核通过才能被访问 |
